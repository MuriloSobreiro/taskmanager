Como funciona a segurança

Este app opera sem Cloud Functions (plano Spark). A autorização é garantida pelas Firestore Security Rules: cada operação verifica o documento /users/{uid} no servidor antes de executar. O campo admin: true só pode ser alterado por outro administrador. Nenhuma lógica de segurança existe somente no front-end.
